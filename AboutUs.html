About Us

The purpose of this site is to help fantasy/scifi worldbuilders and writers.

The site does not aim to create new content, but instead to collect already existing tools and resources to create one extensive archive. The Internet is already full of excellent content to help writers and worldbuilders, but these tools are spread across multitude of different sites. By collecting all these links to one site I want to help all worldbuilders.

Thus the name Worldbuilding through Recycled Goods.
