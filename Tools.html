Tools for Worldbuilding

Campfire

Campfire is a tool for creative writing and keeping your notes in order. This tool is marketed and used by many amateurs and professional writers. Campfire is mainly meant for the creation of a fantasy stories, but it can be applied to many different writing exercises.

Project Forge

Project Forge is based on World Anvil and aims to help new and experienced worldbuilders. The Project has started by creating a game/tool called Realms. Link to download this tool is on their site.
